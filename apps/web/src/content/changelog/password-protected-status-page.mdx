title: Password-protected status page
description: Hide your page from the public.
image: /assets/changelog/password-protected-status-page.png
publishedAt: 2024-05-02
---

You can now protect your status page with a password. This will hide the page
from the public and only allow access to users with the password.

You can enable the password protection from the status page settings.

> This is a Pro feature. Upgrade your account to use this feature.
